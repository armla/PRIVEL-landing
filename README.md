# PRIVEL — Landing Page

Public landing page for [privel.app](https://privel.app) — the private advisory intelligence platform for agents of The Agency Costa Rica.

**Live URL:** [www.privel.app](https://www.privel.app)

---

## Stack

- React 19 + TypeScript
- Tailwind CSS 4
- Vite 7
- Hosted on DigitalOcean App Platform

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Output is in `dist/public/`.

## Deployment

Deployed automatically via DigitalOcean App Platform on push to `main`.

Build command: `pnpm install && pnpm build`
Output directory: `dist/public`
